# EduPlatform - Educational Platform

A modern educational platform built with HTML, CSS, and JavaScript, featuring Materialize CSS framework.

## Features

- Responsive design that works on all devices
- Modern and professional UI using Materialize CSS
- Interactive course catalog with filtering
- Student dashboard with progress tracking
- Contact form with validation
- Interactive programming concept matching game
- Calendar integration for scheduling
- Course progress visualization
- Activity tracking

## Pages

1. Home (index.html)
2. Courses (courses.html)
3. About Us (about.html)
4. Contact (contact.html)
5. Student Dashboard (dashboard.html)
6. Game (game.html)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Materialize CSS Framework
- FullCalendar.js

- Page-specific styles are embedded in each HTML file
- Single element-specific styles use inline CSS
